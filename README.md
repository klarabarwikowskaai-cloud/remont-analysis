Do dalszej analizy warto:
1. Poprosić o szczegółową wycenę materiałów od Sławomira.
2. Umówić się na wizję lokalną z PRO_POZIOM w celu ustalenia ostatecznej ceny.
3. Sprawdzić referencje i portfolio Krzysztofa Górskiego, zwłaszcza dotyczące gwarancji 3-letniej.
